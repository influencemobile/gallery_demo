* install homebrew

  $ /bin/bash -c "$(curl -fsSL https://raw.githubusercontent.com/Homebrew/install/master/install.sh)" 

* install ruby

brew install rbenv
rbenv install 2.5.8
rbenv rehash

* install sqlite
brew install sqlite
