== README

* clone github repo locally

* prerequisites
  * ruby version 2.5.8 or higher

  * sqlite3 (you might already have this)

* setup rails / project
  * gem install bundler
  * bundle install
  * rake db:migrate
  * rake db:seed

* run server

`rails s`

https://localhost:3000/artists
https://localhost:3000/galleries
https://localhost:3000/paintings
